# AI-Based Voice-to-Text EMR Generation

## Overview
This project is an AI-based Voice-to-Text Electronic Medical Record (EMR) Generation System developed during my internship at Bharat Electronics Limited (BEL).

The application converts doctor-patient conversations into structured Electronic Medical Records using Speech-to-Text, Natural Language Processing (NLP), PostgreSQL, and Python.

---

## Features

- Voice Recording
- Speech-to-Text Conversion
- Patient Information Extraction
- Clinical Details Extraction
- Prescription Generation
- PostgreSQL Database Integration
- Search Existing Patient
- PDF Report Generation
- User-Friendly GUI using Tkinter

---

## How to Run

1. Install Python
2. Install PostgreSQL
3. Install the required packages

```
pip install -r requirements.txt
```

4. Update the PostgreSQL connection details.

5. Run

```
python main.py
```
